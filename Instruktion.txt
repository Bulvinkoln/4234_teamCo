Инструкция по работе с Git

В этой инструкции я расскажу о базовых командах Git и о том, как их использовать. Я также использовал форматирование Markdown для улучшения читаемости.

Установка Git

Прежде чем начать работать с Git, вам нужно установить его на свой компьютер. Вы можете загрузить установщик для своей операционной системы с официального сайта Git.

Настройка Git

Перед использованием Git необходимо выполнить несколько настроек. Ниже приведены основные команды, которые необходимо выполнить:

bash
Copy code
git config --global user.name "Ваше имя"
git config --global user.email "Ваш электронный адрес"
Создание репозитория

Репозиторий - это место, где вы храните свой проект. Чтобы создать новый репозиторий, выполните следующие действия:

Откройте терминал и перейдите в папку, в которой вы хотите создать новый репозиторий.
Введите следующую команду:
bash
Copy code
git init
Добавление файлов

Прежде чем вы сможете зафиксировать изменения в Git, вам необходимо добавить файлы, которые вы хотите отслеживать. Чтобы добавить файлы, выполните следующие действия:

Перейдите в корневую папку вашего проекта.
Введите следующую команду, чтобы добавить все файлы в текущей директории:
bash
Copy code
git add .
Фиксация изменений

После того, как вы добавили файлы в Git, вы можете зафиксировать изменения. Чтобы это сделать, выполните следующие действия:

Введите следующую команду, чтобы зафиксировать изменения:
bash
Copy code
git commit -m "Описание ваших изменений"
Клонирование репозитория

Если вы хотите скопировать репозиторий с удаленного сервера на свой компьютер, выполните следующие действия:

Откройте терминал и перейдите в папку, в которую вы хотите скопировать репозиторий.
Введите следующую команду:
bash
Copy code
git clone https://github.com/ваш-username/ваш-репозиторий.git
